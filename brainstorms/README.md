# Brainstorms Output

This directory contains outputs from FleetKit brainstorming sessions.

## Structure

Each brainstorm gets its own folder:

```
brainstorms/
├── 2026-02-18-pricing-strategy/
│   ├── round-1.md
│   ├── round-2.md
│   └── synthesis.md
├── 2026-02-19-go-to-market/
│   ├── round-1.md
│   ├── round-2.md
│   └── synthesis.md
```

Brainstorm files are automatically created when you run the protocol in `workflows/brainstorm.md`.
