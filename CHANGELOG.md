## [1.0.0] — 2026-02-16

### Added

- **5 AI executive agents** — CEO (Chief), COO (Atlas), CRO (Hunter), CTO (Forge), CMO (Echo)
- **Brainstorm protocol** — 3-round structured debate: Positions → Debate → Synthesis
- **Standup protocol** — Automated daily sync with COO synthesis
- **Decision escalation protocol** — Structured framework for unresolved disagreements
- **Fleet dashboard** — Dark-mode HTML dashboard for agent activity
- **Cron configurations** — Morning standup and weekly retrospective
- **Example outputs** — Real brainstorm and standup examples
- **YAML configuration** — Models, temperatures, schedules, budgets in one file
- **Quick start guide** — 5-minute setup from zero to first brainstorm
