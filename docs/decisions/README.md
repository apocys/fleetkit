# Decision Log

This directory contains the decision log for your FleetKit fleet.

## Structure

One file per decision:

```
docs/decisions/
├── 2026-02-18-pricing-strategy.md
├── 2026-02-20-go-to-market-channels.md
├── 2026-02-22-hiring-priorities.md
```

Decision records are created when using the escalation protocol in `workflows/decision.md`.
